# Analisi-dei-Dati--Esame
L'idea di questa repository è quella di creare un posto dove possiamo aiutarci a capire il funzionamento dei vari esercizi che ci saranno all'esame, in modo tale da preparaci 
al messo ad esso. 
La cosa più importante è collaborare. 

## Cartella Esercizi 
Qui troverete i file che riassumono e mostrano come fare gli esercizi delle varei unità. Quindi se dovete aggiungere un file dovete usare questo formato: 
Unità_numero_Es.Rmd

## Cartella Modulo 
In questa cartella invece andrò/ andremo a caricare il preset dei file che useremo al esame, ogni modulo identifica quello che vorrebe usare durante l'esame
